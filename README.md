# PolaScript

数年前(2014年)のコードなので酷いですが今更ながら修正したりライセンス付与するなどしました．

C ライクな手続き型プログラミング言語になるつもりのものです．

```csharp
println("Hello world!");
```

```csharp
String neko = "ねこ";
println(neko + "かわいい");
```

```csharp
Number result = 3 + 4;
println("計算結果は" + result + "です");
```
